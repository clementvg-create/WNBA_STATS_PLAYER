## WNBA_STATS_PLAYER (FRENCH VERSION)

Le dépôt GitHub est organisé de la manière suivante :
 - Dossier /Données --> Données brutes récupérées depuis [BBRef Site Web](https://www.basketball-reference.com/)
 - Dossier /Modèle --> Modèle Excel pour exploiter les données brutes
 - Dossier Racine :
	- STATS_2025.xlsx --> Fichier Excel de la saison 2025 des stats (basiques + avancées)
	- Etc...

Saisons disponibles : 2010-2025 (1997-2009 bientôt disponibles)

# Utilisation

Deux utilisations sont possibles :
 - Avoir Git sur son PC --> Récupération du dépot Git [ici](https://github.com/clementvg-create/WNBA_STATS_PLAYER.git)
 - Télécharger les fichiers Excel (STATS_2025.xlsx ...) en cliquant dessus depuis l'adresse [ici](https://github.com/clementvg-create/WNBA_STATS_PLAYER)

Remarque : Si vous utilisez Git, ne pas faire de modifications sur les fichiers (Commit, Push...), vous pouvez néanmoins Clone/Pull le dépôt à volonté !

## WNBA_STATS_PLAYER (ENGLISH VERSION)

The GitHub repository is organized as follows:
 - Folder /Données --> Raw data retrieved from [BBRef Site Web](https://www.basketball-reference.com/)
 - Floder /Modèle --> Excel template for processing the raw data
 - Root folder:
	- STATS_2025.xlsx --> Excel file for the 2025 season of stats (basic + advanced)
	- Etc...

Available seasons : 2010-2025 (1997-2009 coming soon)

# Usage

Two uses are possible:
 - Having Git on your PC --> Retrieve the Git repository [here](https://github.com/clementvg-create/WNBA_STATS_PLAYER.git)
 - Download the Excel files (STATS_2025.xlsx...) by clicking on them from the address [here](https://github.com/clementvg-create/WNBA_STATS_PLAYER)
 
Note: If you are using Git, do not make any changes to the files (commit, push, etc.). However, you can clone/pull the repository as often as you like!
